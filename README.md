# Projetos feitos durante o curso da disciplina Programação Orientada a Objeto, na linguagem Java, na UFRJ.

Ministrada pelo professor Vinícius Gusmão no período de 2026.1 (meu terceiro período na faculdade).

Aborda inúmeros dos assuntos mais importantes da linguagem e do paradigma de programação, como os tipos primitivos, classes e objetos, herança, packages, modificadores de acesso, polimorfismo, 
composição e agregação, estruturas úteis, unit tests, tratamento de exceções, métodos abstratos e classes abstratas, interfaces, input/output padrão e de/para arquivos,
design patterns, convenções e boas práticas etc.
